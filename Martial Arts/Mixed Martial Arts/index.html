About the data

A data frame for the comprehensive collection of professional mixed martial arts fights has been compiled, encompassing bouts from the Ultimate Fighting Championship, Bellator MMA, and ONE Championship up to August 11, 2021 (the date of data retrieval). This data set comprises information on 10,448 professional MMA fights.

Download data: pro_mma_fights.csv

Data Source: Pro MMA Fights Data Set

Variable Descriptions
Variable Description
url website URL to the event/fight
event_title title of event/fight.
organisation MMA organization that’s promoting the event/fight
date date of event/fight
location location of event/fight
match_nr number of match, in which it take place during the event
fighter1_url website URL to the fighter 1
fighter2_url website URL to the fighter 2
fighter1_name name of fighter 1
fighter2_name name of fighter 2
fighter1_result fighter one’s result
fighter2_result fighter two’s result
win_method how the match was won, like the method (category)
win_details details on how match was won
referee name of the referee for the event/fight
round number of rounds endured before a win
time time taken for someone to win the event/fight
